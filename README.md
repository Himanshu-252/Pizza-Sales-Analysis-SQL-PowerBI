# Pizza Place Sales Analysis

## Project Overview
This project provides a comprehensive analysis of sales data for a fictional pizza shop. The goal is to identify key performance indicators (KPIs), understand customer preferences, and pinpoint peak operational hours to help the business optimize its inventory and staffing.

## Technical Skills Used
- SQL: Joined 4 tables, performed aggregations, and identified peak sales hours.
- Power BI: Built a relational data model and used DAX for KPIs.
- Data Visualization: Created an interactive dashboard for business stakeholders.

## Key Insights
- Revenue Performance: The shop generated a total revenue of $817.86K with an average order value of $38.31.
- Size Matters: Large (L) pizzas are the most popular, contributing to 45.89% of total revenue.
- Top Seller: The Classic Deluxe Pizza is the highest-grossing product, followed closely by the BBQ Chicken Pizza.
