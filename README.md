# Pizza Place Sales Analysis

## Project Overview
This project analyzes a year's worth of sales data from a pizza place to find opportunities for growth and efficiency.

## Technical Skills Used
- **SQL:** Joined 4 tables, performed aggregations, and identified peak sales hours.
- **Power BI:** Built a relational data model and used DAX for KPIs.
- **Data Visualization:** Created an interactive dashboard for business stakeholders.

## Key Insights
- **Busiest Time:** The peak hour for orders is 12:00 PM - 1:00 PM.
- **Top Product:** The Classic Deluxe Pizza contributes the most to total revenue.
- **Size Preference:** Large pizzas account for nearly 45% of total sales.
